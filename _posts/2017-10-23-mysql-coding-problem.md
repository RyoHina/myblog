* content

打开 /etc/mysql/mysql.conf.d/mysqld.cnf
添加：
character_set_server = utf8

打开 /etc/mysql/conf.d/mysql.cnf
添加：
[mysql]
default-character-set = utf8

[client]
default-character-set = utf8

重启mysql服务
service mysql restart

进入mysql检查, 如下所示表示修改好了
mysql> show variables like 'collation%';
